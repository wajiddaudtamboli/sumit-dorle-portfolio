# Sumit Dorle Portfolio

Premium executive portfolio website with dark/light mode, modern UI, and Vercel-ready setup.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Deployment

This project is configured for Vercel static deployment with SPA route support.
